# 📱 每日新闻归纳助手 — 部署指南

> 本项目有 **三种方式** 让家人在手机上使用。推荐方法一（APK 安装包）。

---

## ⚡ 方法一：GitHub Actions 在线构建 APK（推荐）

### 第一步：创建 GitHub 仓库
1. 打开 https://github.com/new
2. 仓库名填 \
ews-digest\，设为 **Private**
3. 点击 **Create repository**

### 第二步：推送代码到 GitHub
打开终端（PowerShell/cmd），运行：
\\\ash
cd D:\codex\新闻归纳助手
git init
git add .
git commit -m "🎉 初始提交"
git remote add origin https://github.com/你的用户名/news-digest.git
git branch -M main
git push -u origin main
\\\

### 第三步：触发构建
1. GitHub 仓库 → **Actions** 标签
2. 左侧 **「构建 Android APK」** → **Run workflow**
3. 等待 15-20 分钟（首次需下载 Android SDK）

### 第四步：下载 APK
1. Actions → 成功的工作流 → **Artifacts**
2. 下载 **新闻归纳助手-APK** → 解压得 .apk
3. 复制到手机安装

> 💡 后续更新只需推送代码重新构建（第二次仅需 2-3 分钟）

---

## 🌐 方法二：Render 云部署（无需 APK，浏览器使用）
1. 推送代码到 GitHub 后
2. 打开 https://render.com 注册 → **New +** → **Blueprint**
3. 连接仓库，Render 自动识别 render.yaml 部署
4. 获得 URL（如 https://news-digest.onrender.com）
5. 手机浏览器打开 → 菜单 → **添加到主屏幕**

---

## 🏗️ 方法三：本地 WSL 构建 APK
\\\ash
# 管理员 PowerShell 安装 WSL
wsl --install -d Ubuntu
# 重启后
wsl -d Ubuntu
cd /mnt/d/codex/新闻归纳助手
bash build-apk-wsl.sh
# APK 在 bin/ 目录
\\\

---

## 🔑 首次使用
打开应用 → 右上角 ⚙️ → 输入 DeepSeek API 密钥（https://platform.deepseek.com）

---

## ❓ 常见问题
- **API 密钥会泄露吗？** 不会，.gitignore 已排除 config.json
- **APK 无法连接？** APK 内置 Python 服务器，如不行请用方法二
- **构建失败？** Settings → Actions → General → 开启 Allow all actions
