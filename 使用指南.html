1 安装 Python

Python 是运行这个程序需要的"环境"，按以下步骤安装：

🪟 Windows 用户

  1. 打开微软应用商店 — 点开始菜单，搜索"Microsoft Store"并打开
  2. 搜索 Python — 在商店右上角搜索框输入 Python
  3. 安装 Python 3.12 — 找到 Python 3.12，点击"获取"或"安装"
✅ 安装完成后验证：按下键盘 Win + R，输入 cmd 回车，在黑窗口中输入 python --version，如果显示 Python 3.12.x 就代表安装成功。

2 获取 DeepSeek API 密钥

AI 需要调用 DeepSeek（深度求索）的服务来帮我们分析新闻。费用非常便宜：分析一次大约 0.01~0.05 元，每天一次一个月不到 2 块钱。

  1. 打开注册页面：https://platform.deepseek.com/（点击链接即可打开）
  2. 注册账号：用手机号或邮箱注册并登录
  3. 进入 API Keys 页面：登录后，在左侧菜单找到 API Keys
  4. 创建密钥：点击 「创建 API Key」 按钮
  5. 复制密钥：会生成一串以 sk- 开头的字符，点旁边的复制按钮
  6. 充值（可选）：新账号通常有赠送额度，不够用了可以在"余额"页面充值，最低 10 元能用很久
⚠️ 保存好密钥：这个密钥就像密码一样，不要发给别人。如果密钥泄露，可以在 DeepSeek 网站上删除重新生成。

3 运行程序

所有文件都在 「新闻归纳助手」 文件夹中，你会看到这几个文件：

📁 新闻归纳助手/
├── 📄 启动新闻助手.bat ← 双击这个文件运行
├── 📄 news_digest.py ← 主程序（不用管）
├── 📄 config.json ← 配置文件（存放密钥）
└── 📄 使用指南.html ← 就是这个页面

▶️ 首次运行步骤

  1. 双击「启动新闻助手.bat」
  2. 程序会自动安装所需的依赖包（只需一次），稍等片刻
  3. 程序会问你要 DeepSeek API 密钥，把上一步复制的密钥粘贴进去，回车
  4. 程序开始抓取新闻并用 AI 分析，等待 1~2 分钟
  5. 分析完成后，浏览器会自动打开，展示今天的新闻榜单！

📅 每天怎么用？

以后每天只需双击「启动新闻助手.bat」，等一两分钟，浏览器就会自动弹出当天的新闻榜单。
什么都不用改，密钥已经保存好了。

❓ 常见问题

Q: 运行时报错"python 不是内部或外部命令"

说明 Python 没有安装成功。重新按照第一步的教程，从微软应用商店安装 Python。

Q: 抓不到新闻 / 全是英文新闻

部分 RSS 源可能被屏蔽或改地址。我会持续更新新闻源，你也可以把这个问题告诉我，我来更新。

Q: AI 分析太慢怎么办？

第一次运行会安装依赖库稍慢。后续运行一般在 30 秒到 2 分钟之间，取决于新闻数量。如果超过 3 分钟，可能是网络问题，重新运行一次试试。

Q: DeepSeek API 怎么充值？

登录 DeepSeek 平台 → 左侧「余额」→ 点「充值」，支持支付宝。充 10 元大概能用几个月。

Q: 我想修改新闻分类或增加分类

可以告诉我，我来帮你改。不改代码也能调整。

Q: 生成的文件在哪里？

就在「新闻归纳助手」文件夹里，文件名是 新闻榜单_2026-06-23.html 这样的格式，双击就能在浏览器打开。
